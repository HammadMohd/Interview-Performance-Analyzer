Audio Module – Interview Performance Analyzer

Overview

The Audio Module is responsible for extracting and analyzing audio information from interview videos.

Objectives

- Extract audio from interview videos
- Preprocess audio signals
- Extract speech-based features
- Generate structured feature data for further analysis

Features to Extract

- MFCC features
- Pitch
- Energy
- Zero Crossing Rate
- Speech Rate
- Pause Duration
- Silence Ratio
- Transcript information

Technologies Used

- Python
- Librosa
- PyTorch
- FFmpeg
- Hugging Face Datasets
- OpenAI Whisper (if required)
